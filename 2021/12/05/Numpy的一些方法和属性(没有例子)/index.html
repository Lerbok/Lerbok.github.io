Numpy的一些方法和属性(没有例子)

数据分析-Numpy
python
Publish Date:   2021-12-05

一.Numpy 中的一维数组创建方法

二.arange()函数

  1. arange()函数创建数值范围并且返回数组的对象

    start : 开始值，默认值是0， 包含开始值

    stop : 结束值，不包含结束值

    step : 补偿， 默认值是1，该值可以为负数

    dtype: 数组元素的类型

三.linspace()函数

  1. 创建等差数组

    num: 设置生成元素的个数

    endpoint: 设置是否包含结束值 默认为包含 即true

    retstep: 设置是否返回步长(即公差), 默认为false, 当值是True的时候，返回的是二元组，包括数组和步长

四. logspace()函数

  1. 创建等比数组

    start : 开始值，值为base**start

    stop: 结束值，值为base**stop

    base : 底数

五. 数组的轴

  1. 数组的转置 多元数组会有0 1 两个轴

六. ones()函数

  1. 根据指定的形状和数组类型生成全部为1的数组

    shape : 数组的形状

七. zero()函数

  1. 根据指定的形状和数据的类型生成全部为0的数组

    shape : 数组的形状

八. full()函数

  1. 根据指定的形状和数据类型生成数组，并且使用指定的数组填充

    fill_value : 指定填充的数据

九. identity()函数

  1. 创建单位矩阵(即对角线元素为1其他元素为0

    n : 数组的形状

十. 连接数组

  1. concatenate()函数

    a1,a2 是要连接的数组 注意，除了指定的轴外 其它轴元素个数必须相等

    axis 是指沿指定轴的索引，默认为0轴

  2. vstack()函数

    沿垂直堆叠多个数组， 相当于concatenate()函数axis=0 的情况

    ps :1轴上的元素个数要相同

  3. hstack()函数

    沿着水平堆叠多个数组，相当于concatenate()函数 axis= 0 的情况

    0轴上的元素个数一定要相同

十一. 分割函数

  1. split()

    该函数值沿着指定的轴分割多个数组

    ary: 要被分割的数组

    indeces_or_sections : 是一个整数或者数组，如果是整数就使用该数平均分割； 如果是数组，则为沿指定轴做切片操作

    axis : 指轴的分割方向，默认为0轴

  2. vsplit()函数

    该函数指沿垂直方向分割数组，相当于split()函数axis=0的情况

​

  1. hasplit()函数 相当于split()函数 axis=1

十二. 随机数函数

  1. rand() 返回[0.0, 1.0]的随机浮点数 即大于等于0.0， 且小于1.0的随机浮点数

    d0, d1…, dn 表示数组的形状

  2. randint()函数 返回[low,high]的随机整数，如果high省略，则会返回[0,low]的随机整数

    size表示数组的形状 (元组)

  3. normal()函数返回正态分布随机数

    loc 表示平均值

    scale 表示标准差

  4. randn()函数返回标准正太分布随机数，即平均数为0，标准差1的正态分布随机数

十三. 排序函数

  1. sort()函数按照轴对数组进行排序，即轴排序

    a 表示要排序的数组

    axis 表示要排序的轴索引，默认是-1，表示最后一个轴，速度最快；mermesort: 归并排序; heapsort : 堆排序

    order 表示排序字段

  2. argsort()函数按照轴对数组经行排序索引，即轴排序索引

​

十四. 聚合函数

  1. 求和

    1. 使用Numpy中的sum()函数

    2. 使用Numpy中的nansum()函数,该书忽略NAN

    3. 使用数组对象的sum()方法

  2. 求最大值

    1. 使用amax()函数

    2. 使用nanmax()函数

    3. 使用数组对象的max()方法

  3. 求小值

    1. 使用amin()函数

    2. 使用nanmin()函数

    3. 使用数组对象的min()方法

十四.数组的保存

  1. save()该函数可以将一个数组保存至后缀名为“.npy”的二进制文件中

    file 表示文件名/文件的路径

    arr 表示要储存的数组

    allow_pickle 为布尔值,表示是否使用pickle来保存数组对象

    fix_imports 为布尔值,表示是否允许在Pyhton2中读取python3保存的数据

  2. savez()该函数可以将一个数组保存至未压缩的后缀名为“.npz”的二进制文件中

  3. savez_compressed()函数可以将多个数组保存到压缩的后缀名为“.npz”的二进制文件中

十五.load()函数

  1. 读取 .npy 和.npz文件中的数组

    mmap_mode表示内存的映射模式，即在读取较大的Numpy数组时候的模式，默认情况下是None
